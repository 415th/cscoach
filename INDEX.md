## ðŸ“ž Support Checklist

### If Something Doesn't Work

**Language switcher not working?**
```
1. Open F12 console
2. Click language button
3. Check for errors
4. Verify .lang-btn exists
5. Check translations.ru object
```

**Mobile menu not appearing?**
```
1. Resize browser <768px
2. Should see hamburger icon
3. Click to toggle
4. Check media query in CSS
```

**Form not submitting?**
```
1. Check console (F12)
2. Verify form ID
3. Check validation logic
4. Ensure required fields filled
5. Look for error messages
```

**Colors wrong?**
```
1. Check CSS variables
2. Find :root { in styles.css
3. Verify color values
4. Clear browser cache
5. Hard refresh (Ctrl+Shift+R)
```

**Images not showing?**
```
1. Check file path
2. Verify file exists
3. Check permissions (755)
4. Try absolute path if needed
5. Check console for 404
```
