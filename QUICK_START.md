### Issue: Language switcher not working?
```
1. Open browser console (F12)
2. Check for errors
3. Verify LanguageManager is running
4. Reload page (Ctrl+R)
```

### Issue: Mobile menu not appearing?
```
1. Resize browser to <768px
2. Should see hamburger icon
3. Click to toggle menu
4. Check CSS media queries in styles.css
```

### Issue: Form not submitting?
```
1. Open console (F12)
2. Check for validation errors
3. Ensure phone is in form
4. Try again
5. Check browser console message
```

### Issue: Styles not applying?
```
1. Clear browser cache (Ctrl+Shift+Delete)
2. Hard refresh (Ctrl+Shift+R)
3. Check styles.css is linked in HTML
4. Check for CSS syntax errors
```
